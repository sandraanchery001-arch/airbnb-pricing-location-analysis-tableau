# 🏡 Airbnb Pricing & Location Analysis | Tableau

## 📌 Project Overview

This project analyzes Airbnb listings to understand pricing trends, room type distribution, host characteristics, and location-based patterns. Using Tableau, the dashboard transforms listing data into meaningful business insights that can help hosts optimize pricing strategies and improve occupancy potential.

---

## 🎯 Business Objectives

- Analyze Airbnb pricing across different room types.
- Identify the most common listing categories.
- Explore the relationship between reviews, availability, and pricing.
- Understand host and location trends.
- Support data-driven pricing and listing decisions.

---

## 📂 Dataset Information

- **Domain:** Hospitality / Travel
- **Dataset Source:** Kaggle
- **Visualization Tool:** Tableau
- **Data Preparation:** Cleaned and transformed before dashboard development.

---

## 📊 Dashboard Features

- Listing Distribution by Room Type
- Average Price by Room Type
- Price Distribution Analysis
- Host Listing Analysis
- Review Analysis
- Interactive Filters for Room Type, Location, and Availability

---

## 🔍 Key Business Insights

- Entire homes command the highest average prices.
- Private rooms make up a significant share of listings.
- Pricing varies considerably across room types.
- Listings with stronger reviews tend to attract greater customer interest.
- Location and room category strongly influence pricing.

---

## 💼 Business Recommendations

- Optimize pricing based on room category and market demand.
- Encourage hosts to improve guest experiences to increase reviews.
- Focus marketing efforts on high-performing listing categories.
- Regularly monitor pricing trends to remain competitive.
